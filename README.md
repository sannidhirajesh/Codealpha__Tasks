# Stock Portfolio Tracker (Python)

This is a simple Python program that calculates total investment value based on user input and predefined stock prices.

## Features
- User inputs stock name and quantity
- Calculates total investment value
- Displays portfolio summary
- Saves data to a file

## Concepts Used
- Dictionary
- Loops
- Conditional statements
- File handling

## How to Run

1. Install Python
2. Run the file:

python Stack_portfolio_tracker.py
