# 📧 Email Extraction Automation Script

## 📌 Description
This project is a Python-based automation script that extracts email addresses from a text file using regular expressions.

## ⚙️ Features
- Reads data from a file
- Extracts valid email addresses
- Displays output on screen
- Saves emails to a file

## 🛠️ Technologies Used
- Python
- Regex

## ▶️ How to Run
1. Place input text in `input.txt`
2. Run the program:
   python email_extractor.py
3. Output will be saved in `emails.txt`
