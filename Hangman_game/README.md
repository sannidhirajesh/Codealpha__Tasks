# Hangman Game (Python)

This is a simple text-based Hangman game built using Python.

## Features
- Random word selection
- 6 incorrect guesses allowed
- Displays guessed letters
- Console based game

## Concepts Used
- Python lists
- Loops
- Conditional statements
- Random module

## How to Run

1. Install Python
2. Run the file:

python hangman.py
